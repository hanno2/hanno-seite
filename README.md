# Hanno's Portfolio Webseite

Eine moderne, responsive Portfolio-Webseite erstellt mit Bootstrap 5.

- 📱 Vollständig responsive Design
- 🎨 Bootstrap 5 Framework
- ✨ Moderne CSS-Animationen
- 📧 Kontakt-Sektion
- 🖼️ Profilbild-Integration

## Technologien

- CSS3
- Bootstrap 5.3.0
- Bootstrap Icons
